# TrendRadar 简明配置手册 (小白专用)

> 恭喜你部署成功！
> 以前修改配置需要改文件、重启容器，很麻烦。
> **现在，我们为您准备了专属的"遥控器"！**

---

## 🔥 核心推荐：使用"遥控器"管理 (最简单)

不需要懂代码，不需要找文件，直接在黑框框里输入这一行命令：

```bash
docker exec -it trend-radar python manage.py
```

你会看到一个像菜单一样的界面，通过输入数字 `1`, `2`, `3` 就能完成所有操作！

### 🎮 这个"遥控器"能做什么？

1.  **⏱️ 修改抓取频率 (定时任务)**
    *   觉得半小时一次太吵？选 `1`，直接改成"每小时"或"每天"。
    *   **注意**：改完时间后，记得按提示重启一下容器 (`docker restart trend-radar`)。

2.  **📝 管理关键词 (你想看什么)**
    *   选 `2` 进入关键词管理。
    *   **支持分组管理**：你可以创建"科技组"、"娱乐组"，把关键词分类存放。
    *   **批量操作**：一次性添加好几个词，或者一次性删除好几个。
    *   **立即生效**：改完下次抓取就生效，不用重启！

3.  **📺 管理监控平台 (你想去哪看)**
    *   选 `3` 进入平台管理。
    *   **一键开关**：不想要抖音热搜？不想看贴吧？直接输入对应序号，一键禁用！
    *   **批量管理**：可以一次性开启或关闭多个平台。

4.  **▶️ 立即运行**
    *   等不及定时任务？选 `5`，立刻抓取一次最新热搜并发给你。

## 📂 进阶：手动修改配置文件 (如果你喜欢动手)

如果你更喜欢直接修改文件，所有配置都在你映射的 `trendradar/config` 文件夹里。

### 1. 修改通知设置 (config.yaml)
想换飞书、钉钉或者邮件通知？
*   使用管理工具选 `4`，或者直接编辑 `trendradar/config/config.yaml`。
*   找到 `notification` 部分填入你的 Webhook 地址。

### 2. 关键词文件 (frequency_words.txt)
*   这是一个纯文本文件。
*   一行一个词。
*   **小技巧**：你可以用空行把不同类型的词隔开，管理工具会自动识别成分组哦！

```text
人工智能
ChatGPT

(这里是个空行，下面是另一组)
特斯拉
马斯克
```

### 3. 开启/关闭平台
*   推荐使用管理工具 `3. 📺 管理监控平台`。
*   如果手动修改：在 `config.yaml` 的最下方 `platforms` 列表里。
    *   **开启**：确保行前面没有 `#` 号。
    *   **关闭**：在行前面加 `#` 号注释掉。

## ❓ 常见问题

**Q: 我改了配置怎么没反应？**
A: 修改**定时任务** (`CRON_SCHEDULE`) 后，必须重启容器才能生效：
```bash
docker restart trend-radar
```
修改**关键词**、**平台**和**通知配置**，通常下次运行时会自动生效。

**Q: 怎么看运行日志？**
A: 输入：`docker logs -f --tail 100 trend-radar`
